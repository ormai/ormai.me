About the /colophon page.

This site is a work in progress and will remain as such forever. All the changes made to this website are tracked with Git, the distributed versioning system. The source of this website is licensed under AGPL-3.0-or-later and can be found here and here (mirror).

This site aims to be simple, in the sense that I avoid using a lot of third party tools and frameworks. I stick with standard HTML and CSS and I might include some JavaScript in the future if I ever want to have some dynamic pages, that might be cool. The build pipeline is thin and defined in a brief shell script that calls various tools. To write this site I directly edit the HTML files in a text editor, the one I use should be mentioned in /uses. I think this gives me more control on the content of each page, as opposed to using a static site generator.

Just HTML and CSS can bring you far enough. The most recent developments like the addition of the <dialog> element are promising. I also try not to make each visitors download external fonts, as the device itself should already have them. Lastly I strive to have cool URIs, and you should too because it's very annoying when I bookmark your site and then I am met with a 404 error page.

Other than be standard compliant this site aims to be accessible and respect its visitors. In this regard improvements are probably necessary, since I have little knowledge in this field. Feedback is very welcome through the channels provided elsewhere in this site.

As a principle, this website doesn't have tracking cookies or code for analytical purposes. At the time of writing, only one cookie is set as a countermeasure for aggressive web scrapers, but that might change in the near feature, as the AI hellscape we live in evolves rapidly and continuously. Usually I bind the Nginx access and error logs to /dev/null, but recently this changed, as I am monitoring them for anomalous requests from bots.

This site is served via Nginx from my home by a mini computer with an Intel® Celeron® J4105 and 8 GB of memory. I use the server to self-host various services such as FreshRSS at feed.ormai.me, a Forgejo instance at git.ormai.me, a CalDav/CardDav service with Radicale at ormai.me/radicale, and a Matrix server with Continuwuity. The setup is pretty home-made, with the server being behind NAT and without a static IP address (I use a custom script for dynamic DNS).
